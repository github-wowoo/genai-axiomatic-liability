# GenAI Distributed Liability Axiomatic System
Official open-source repository for the paper **"The Responsibility Gap in Generative AI: An Axiomatic Analysis of Distributed Liability and Governance Mechanisms"**, published in *Sustainability* (MDPI).

## Overview
This repository provides a formally validated, fully replicable axiomatic framework for resolving the systemic responsibility gap in generative AI (GenAI) governance. The framework is anchored in the UN Sustainable Development Goals (SDGs), with full logical validation, robustness testing, and real-world case validation across global jurisdictions.

### Core Contributions
1. **Formally Validated Axiom System**: The first logically consistent, independent, and complete axiomatic framework for GenAI distributed liability, with 5 core axioms and explicit priority rules.
2. **SDG-Aligned Governance Framework**: Embeds sustainable development as the overarching principle of GenAI liability design, with a quantifiable measurement framework for SDG 9, SDG 10, SDG 16, and SDG 17 outcomes.
3. **Fully Replicable Research**: All code, supplementary materials, and validation protocols are open-sourced to enable full replication of the paper's results.
4. **Global Applicability**: Validated across common law and civil law jurisdictions, developed and developing economies, with specialized rules for open-source GenAI models.
